# Evidence Foundry

Evidence Foundry is an experimental lab for synthetic evidence generation, causal investigation systems, and trustworthy reasoning experiments.

The project explores how reasoning systems can investigate observable evidence, form explicit claims and beliefs, and be evaluated against hidden ground truth.

Areas of exploration include:
- causal reasoning
- evidence-based investigation
- observable vs hidden reality
- grounded evaluation
- belief tracking
- trustworthy analytical reasoning


## Core Idea
```text
hidden world
    ↓
observable evidence
    ↓
reasoning system
    ↓
claims and beliefs
    ↓
evaluation against ground truth
```

## Initial Architecture
```text
World Ontology
    ↓
Simulated Case
    ↓
Observed Ontology + Evidence
    ↓
Reasoner
    ↓
Beliefs / Claims
    ↓
Evaluation vs Ground Truth
```
